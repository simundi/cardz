# Cardz — Delegation Poker (Remote)

This project is a learning exercise to explore [OpenSpec](https://github.com/anthropics/openspec), a spec-driven development workflow. The app itself is a remote-friendly Delegation Poker tool, but the primary goal is experimenting with OpenSpec as a way to structure and drive development.

## Running the app

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Other commands

```bash
npm run build      # Production build
npm run lint       # Run ESLint
npm run preview    # Preview production build
```
